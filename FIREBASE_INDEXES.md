# Создание индексов в Firestore

## Быстрое решение

Firebase требует индексы для некоторых запросов. Есть два способа:

### Способ 1: Автоматическое создание (РЕКОМЕНДУЕТСЯ)

1. Откройте ссылку из ошибки в браузере
2. Firebase автоматически создаст нужный индекс
3. Подождите 1-2 минуты пока индекс создастся
4. Готово!

### Способ 2: Ручное создание

1. Откройте [Firebase Console](https://console.firebase.google.com/)
2. Выберите проект `apevault`
3. Перейдите в **Firestore Database** → **Indexes**
4. Нажмите **"Create Index"**
5. Заполните:
   - Collection ID: `dayStatuses`
   - Fields to index:
     - `userId` (Ascending)
     - `date` (Ascending)
6. Нажмите **"Create"**
