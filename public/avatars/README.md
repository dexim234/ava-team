# Папка для аватарок участников

Поместите сюда фотографии участников команды:

- `artyom.jpg` - фотография Артёма
- `adel.jpg` - фотография Адели
- `kseniya.jpg` - фотография Ксении
- `olga.jpg` - фотография Ольги
- `anastasia.jpg` - фотография Анастасии

Формат: JPG, PNG или любой другой веб-формат.
Рекомендуемый размер: 200x200 пикселей или больше (изображение будет автоматически обрезано до круга).
